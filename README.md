# Hydrostatic pressure as a predictor of star formation in nearby galaxies

The objective of this project is to show several data analysis methods used in my study of the relation between the hydrostatic pressure and star formation rate density on local scales in galaxies observed as part of the ACA-EDGE survey, where I compare said relation to the ones previously established between the stellar mass density, molecular gas mass density, and star formation rate density. The csv files contain the data used, which can be merged together into a single database using the `Source_id` column, and the Jupyter noteboook `database_analysis.ipynb` showcases the analysis, including OLS fits to the main relations, and the use of machine learning to select the most important feature for predicting star formation rates.

# Presión hidrostática como un predictor de formación estelar en galaxias cercanas

El objetivo de este proyecto es mostrar varios métodos de análisis de datos usados en mi estudio de la relación entre la presi{on hidrostática y la razón de formación de estrellas en escalas locales en las galaxias observadas como parte del sonde ACA-EDGE survey, donde comparé dicha relación con las ya establecidas relaciones entre la densidad de masa estelar, la densidad de masa de gas molecular y la razón de formación de estrellas. Los archivos csv contienen los datos usados, los cuales pueden ser unidos en una única base de datos usando la columna `Source_id`, y el cuaderno de Jupyter `database_analysis.ipynb` muestra todo el análisis, incluyendo ajustes de mínimos cuadrados a las relaciones de interés, así como el uso de un modelo de aprendizaje automático para seleccionar las características más importantes para predecir valores de razón de formación de estrellas.
